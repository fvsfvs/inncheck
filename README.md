### Развертывание проекта
```
git clone https://github.com/fvsfvs/inncheck.git
```
```
cd inncheck
```
```
docker compose up -d --build
```
```
docker compose exec php composer install
```
```
docker compose exec php php bin/console doctrine:migrations:migrate
```

Создать файл .env.local, добавить туда DADATA_API_TOKEN= здесь указать свой токен из лк Dadata

### Запуск проекта

```
docker compose up -d
```
### Поиск организаций по ИНН
http://localhost:8080/api/organizations/inn/{inn}

### Принцип работы

При вызове http://localhost:8080/api/organizations/inn/{inn}
программа получает данные по всем организациям с данным инн (их может быть несколько с учетом закрытых ИП).
Для организаций выдача ограничена только главной организацией - без учета филиалов.
Берется только основной ОКВЭД, т.к. на бесплатном тарифе передается только он.

При запросе в Dadata данные сохраняются в БД.
В config/services.yaml прописывается параметр ```organization.ttl_days: 30``` - сколько дней мы доверяем данным из базы, в течение этого времени данные по уже запрошенному ИНН будут браться из БД без запроса в DaData.

При обновлении ранее полученных данных организации могут быть удалены через soft delete (при наличии их в базе, и остутствии в ответе Dadata) или восстановлены.
Хотя предполагается, что организации из реестра не удаляются - это сделано на случай, если Dadata в какой-то момент может передать ошибочные данные, чтобы синхронизация с получаемыми данными всегда сохранялась.

Организации из базы синхронизируются с Dadata по ОГРН - предполагается, что он уникален для каждой организации.

### Описание ответа

message - сообщение при ошибке
data - массив организаций

#### Содержание data

| Поле      | Тип | Описание |
|-----------|------|---------|
| inn       | string | Запрошенный ИНН |
| ogrn      | string | ОГРН |
| name      | string| Сокращённое название с организационно-правовой формой |
| okved     | string | Основной ОКВЭД |
| okvedType | string | Год версии справочника ОКВЭД |
| isActive  | boolean | Является ли организация действующей |
